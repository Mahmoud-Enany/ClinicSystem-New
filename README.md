# ClinicSystem 🏥

A full-stack clinic management system built with ASP.NET Core 10 and Angular 21.

## Tech Stack
**Backend:** ASP.NET Core 10, EF Core, SQL Server, JWT Auth, AutoMapper  
**Frontend:** Angular 21, Bootstrap 5, TypeScript, Reactive Forms, Signals

## Features
- Role-based panels (Admin, Doctor, Patient)
- Online appointment booking
- Medical records management
- Stripe payment integration
- JWT authentication & authorization
- Admin dashboard with live statistics
